# 首笔受控实单 运行手册（2026-06-11）

查询：`market.duckdb` bars 表近 60 交易日，按 日收益标准差升序，过滤
沪深主板 / 收盘价 3-8 元 / 日均成交额 ≥3 亿（close×volume×100，volume 单位为手）。

| symbol | 收盘(库内最后日) | 60日波动 | 日均额 |
|---|---|---|---|
| **601006.SH 大秦铁路** | 5.16 | **0.53%** | 5.9 亿 |
| 600905.SH 三峡能源 | 4.09 | 0.61% | 5.1 亿 |
| 601816.SH 京沪高铁 | 5.15 | 0.64% | 6.0 亿 |

**选定 601006.SH 大秦铁路**：全场最低波动（与 P0 判决中最接近过线的 F04
低波动因子同向）、流动性充足保证贴卖一限价秒成、一手约 ¥520 资金占用最小。

## 前置条件

- **QMT 以极简模式登录**（行情 + 交易两条链路都依赖它；
  投研版仅行情，xttrader connect 会返回 -1——已实测）
- 账号无需手工配置：CLI 按 env `QMT_ACCOUNT_ID` → trading.yaml → 交易端枚举自动解析

## 执行命令

```bash
WIN_PYTHON="/mnt/c/Users/11492/.conda/envs/goldenhandquant/python.exe"
$WIN_PYTHON -m src.interfaces.cli.quant order buy --symbol 601006.SH --lots 1 --yes
```

执行链：订阅 tick（3s 超时回退快照）→ 六重检查（主板/时段/报价/涨跌停带/
金额≤1500/资金）→ 限价 min(卖一, 最新价×1.002) → 提交 → 2s 轮询至终态 →
审计 JSON 落 `data/trade_logs/`（账号脱敏）。
