## 执行命令

```bash
WIN_PYTHON="/mnt/c/Users/11492/.conda/envs/goldenhandquant/python.exe"
$WIN_PYTHON -m src.interfaces.cli.quant order buy --symbol 601006.SH --lots 1 --yes
```

执行链：订阅 tick（3s 超时回退快照）→ 六重检查（主板/时段/报价/涨跌停带/
金额≤1500/资金）→ 限价 min(卖一, 最新价×1.002) → 提交 → 2s 轮询至终态 →
审计 JSON 落 `data/trade_logs/`（账号脱敏）。
