算法交易

大额订单自动拆分，减少市场冲击。支持三种算法策略：

TWAP — 时间加权平均价格

将大额订单按时间均匀拆分执行：

●python
from src.domain.trade.services.algo_strategies.twap_strategy import TWAPStrategy

strategy = TWAPStrategy(
    total_volume=10000,
    num_slices=10,
    interval_seconds=60,
)
slices = strategy.generate_slices()

VWAP — 成交量加权平均价格

根据历史成交量分布拆单，高峰时段多拆：

●python
from src.domain.trade.services.algo_strategies.vwap_strategy import VWAPStrategy

strategy = VWAPStrategy(
    total_volume=10000,
    volume_profile=historical_volumes,
)
slices = strategy.generate_slices()

冰山单 — 隐藏真实意图

只显示部分数量，成交后自动补充挂单：

●python
from src.domain.trade.services.algo_strategies.iceberg_strategy import IcebergStrategy

strategy = IcebergStrategy(
    total_volume=50000,
    visible_volume=1000,
    price=100.0,
)
💡
算法交易通过 IAlgoTrader 接口定义，具体实现在 infrastructure 层。可与实盘交易引擎集成。

组合优化

科学化的资金分配，提升风险调整后收益。支持三种优化模型：

均值-方差优化

经典 Markowitz 模型，最大化夏普比率：

●python
from src.domain.portfolio.services.optimization.mean_variance_optimizer import MeanVarianceOptimizer

optimizer = MeanVarianceOptimizer()
result = optimizer.optimize(
    expected_returns=returns,
    covariance_matrix=cov_matrix,
    risk_free_rate=0.03,
)
# result.weights — 最优权重
# result.sharpe_ratio — 夏普比率

Black-Litterman 模型

市场均衡收益 + 主观观点 = 后验收益估计：

●python
from src.domain.portfolio.services.optimization.black_litterman_optimizer import BlackLittermanOptimizer

optimizer = BlackLittermanOptimizer()
result = optimizer.optimize(
    market_caps=market_caps,
    views=[("600519.SH", 0.15, 0.8)],
)

风险预算

按风险贡献分配权重，实现风险平价：

●python
from src.domain.portfolio.services.optimization.risk_budget_optimizer import RiskBudgetOptimizer

optimizer = RiskBudgetOptimizer()
result = optimizer.optimize(
    covariance_matrix=cov_matrix,
    risk_budgets=[0.25, 0.25, 0.25, 0.25],
)
