🚀 快速开始

环境要求

  • Python 3.13+ (WSL conda: goldenhandquant)
  • QMT 客户端 (MiniQMT) 已启动，路径 C:\QMT\userdata_mini
  • Windows Python: /mnt/c/Users/11492/.conda/envs/goldenhandquant/python.exe

安装

conda create -n goldenhandquant python=3.13 -y
conda activate goldenhandquant
pip install -e ".[dev,api,ml]"

❓ 常见问题

QMT 连接失败？

确保 MiniQMT 已启动，运行 python scripts/test_qmt_connection.py。xtdata 自动连接 127.0.0.1:58610。

WSL 中如何用 xtquant？

必须用 Windows Python:

/mnt/c/Users/11492/.conda/envs/goldenhandquant/python.exe -m src.interfaces.cli.fetch_quote -s 600519.SH

quant ml-train 报 "unrecognized arguments"？

已修复。直接用: python -m src.interfaces.cli.quant ml-train --symbols 600519.SH --start-date 2024-01-01 --end-date 2024-12-31

北向资金/龙虎榜/行业板块不可用？

xtdata 无这些专用接口，命令会优雅降级。建议用第三方数据源。

ML 数据集如何构建？

直接运行 ml-train，数据集不存在时自动从 QMT 构建。

输出格式？

  "success": true,
  "data": { ... },
  "error": null,
  "timestamp": "2026-06-01T01:06:54+08:00"
}
